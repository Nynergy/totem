You can use inline footnotes to hide information from the reader unless they click on the little footnote superscript indicator. They look like this. Now you can read this information. Click the indicator again to hide it.

You can use footnotes to stow away extra text, or any arbitrary HTML you want to throw in there as well.

A Note About External Links

If you include links to external sites in your website, then totem will expect you to "mark" them as external. What you should do is the following:

<a href="https://example.com">Link Text</a>

Should become:

<a { EXT_LINK } href="https://example.com">Link Text</a>

Except, you don't want those spaces between the curly braces ({,}) and the EXT_LINK. I'm just doing that here otherwise it will get replaced when I render this post.

During the build process, that marker will be replaced with target="_blank" rel="noopener noreferrer" which guarantees that the link will open in a new tab instead of the current one.

You can test it out by following this link.

NOTE: If you forget to include the EXT_LINK marker, totem will complain about that specific link during the build process, so you can go find it and add the EXT_LINK marker.
