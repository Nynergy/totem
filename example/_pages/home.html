Locus Temporarius is an example website built with totem.

Things To Do Here

Click on links. Follow them. Explore the site.

If you want to know why something is styled the way it is, take a look at the file _style/style.css and make changes if desired.

Pay special attention to the following CSS:

section, section:target ~ #home, #home:has(~ section:target) { display: none; } #home, section:target { display: block; }

This is what allows the site to show/hide pages based on what the user has selected. Removing it may lead to unintended site behavior.
